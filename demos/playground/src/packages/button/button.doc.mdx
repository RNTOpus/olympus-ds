### Como usar:
> [Instalar a biblioteca](/docs/introduction--page) e depois:

#### no js/ts:
```js dark
import '@olympus/button';
```
#### no html:

```html dark
<ods-button></ods-button>;
```
[ver outros exemplos em html](#outros-exemplos-de-uso-no-html)

#### no css:
> **Importante:** Para usar os tokens do **Olympus DS** será necessário importar o arquivo **index.css**, conforme abaixo:
- **OBS.:** É de responsabilidade da aplicação setar os tokens, desta forma é mais fácil para customizá-los.
[Ler mais sobre os design tokens do ODS]().

```css dark
  @import "~@olympus/tokens/index.css";
```

## Outros exemplos de uso no html

```html dark

<ods-button></ods-button>
<!-- Color danger|warning|success -->

<ods-button disabled></ods-button>
<!-- Oulined -->
<ods-button outlined></ods-button>
<!-- Fully -->
<ods-button fully></ods-button>;
```

## Angular
> No **Angular** é necessário adicionar `schemas: [CUSTOM_ELEMENTS_SCHEMA]` no **module**, [ver mais](/docs/introduction--page#angular).

```typescript dark
import '@olympus/button';

...

export class MyComponent {

  buttonState = "primary"
  buttonSize = "medium"
  buttonText = "Botão no angular"
  buttonDisabled = false

  onClick() {
    this.buttonText = "Botão clicado"
    this.buttonDisabled = true
  }
  ...
}
```

```html dark
<ods-button
  [state]="buttonState"
  [size]="buttonSize"
  [disabled]="buttonDisabled"
  outlined
  fully
  (click)="onClick()"
>
  {{buttonText}}
</ods-button>
```

## Svelte
> No **Svelte** não precisa de configuração prévia.

```javascript dark
  <script>
    import "@olympus/tokens/index.css";
    import '@olympus/button';
    export let buttonText = "Botão no Svelte";
    export let state = "primary"
  </script>
```

```html dark
  <ods-button
    state={state}
    on:click="{() => state = 'success'}">
      {buttonText}
  </ods-button>
```

## Vue
> No **Vue** é necessário adicionar `isCustomElement` no **compilerOptions no vite.config.js**, [ver mais](/docs/introduction--page#vue).

```javascript dark
<script setup>
  import { reactive } from 'vue'
  import '@olympus/button';
  const button = reactive({
    label: 'Botão no VUE',
    state: 'primary',
    disabled: false,
    outlined: true
  })
  function clickEvent(event) {
    console.log('CLICADO!');
    button.state = 'success'
  }
</script>
```

```html dark
<template>
  <ods-button
    :state="button.state"
    :disabled="button.disabled"
    :outlined="button.outlined"
    @click="clickEvent"
  >
    {{button.label}}
  </ods-button>
</template>
```

```css dark
<style>
  @import "@olympus/tokens/index.css";
</style>
```

## Estilo customizado
É possível customizar o estilo do botão usando `::part()`, veja abaixo:<br />[Saiba mais](https://developer.mozilla.org/en-US/docs/Web/CSS/::part)

```css dark
ods-button::part(button) {
  width: 100px;
  --ods-button-border-radius: 50px;
}
```
