## Instalação

Para instalar um pacote do `@olympus` no seu projeto:

com [npm](https://www.npmjs.com/), rode:

```bash dark
npm install @olympus/{package-name}
```

se preferir [yarn](https://yarnpkg.com/en/), rode:

```bash dark
yarn add @olympus/{package-name}
```

### Uso:
```js dark
import '@olympus/{package-name}';
```

> **Importante:** Para usar os tokens do **Olympus DS** será necessário importar o arquivo **index.css**, conforme exemplo mais abaixo:
- **OBS.:** É de responsabilidade da aplicação setar os tokens, desta forma é mais fácil para customizá-los.<br />
[Saiba mais sobre a biblioteca de Tokens do Olympus DS]().
```css dark
  @import "~@olympus/design-tokens/index.css";
```

## Angular
> Para usar **Web Components** no Angular.<br />Adicione **CUSTOM_ELEMENTS_SCHEMA** no `@NgModule.schemas`, conforme exemplo abaixo:

```js dark
import { NgModule, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';
...
@NgModule({
  ...
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
  ...
})
export class AppModule { }

```
> Exemplo de uso do componente `button` no **ANGULAR**:

```typescript dark
import '@olympus/button';

@Component({
  selector: 'my-component',
  template: `
    <ods-button
      [state]="buttonState"
    >
      {{buttonText}}
    </ods-button>
  `,
  styleUrls: ['./my.component.css']
})
export class MyComponent {
  buttonState = "primary"
  buttonText = "Botão no angular"
}
```

## React
> Não é necessário configuração para usar **Web Components** no React, mas:
- React só pode passar tipos de dados primitivos para atributos HTML (não arrays ou objetos)
- O React não pode escutar eventos do Web Component, então você deve anexar manualmente seus próprios handlers.<br />
**A versão @experimental do React oferece suporte completo para web components.**

## Svelte
> Não é necessário configuração para usar **Web Components** no Svelte, veja o exemplo de uso do componente `button`:

```javascript dark
  <script>
    import "@olympus/design-tokens/index.css";
    import '@olympus/button';
    export let label = "Botão no Svelte";
    export let state = "primary"
  </script>
  <ods-button
    state={state}
    on:click="{() => color = 'success'}">
      {label}
  </ods-button>
```

## Vue
> Para usar **Web Components** no Vue.<br />Adicione **isCustomElement** no `compilerOptions`, conforme exemplos abaixo:<br />[Documentação oficial](https://vuejs.org/guide/extras/web-components.html)

**In-Browser config:**
```js dark
app.config.compilerOptions.isCustomElement = (tag) => tag.includes('-')
```

**Vite config:**
```js dark
import vue from '@vitejs/plugin-vue'
export default {
  plugins: [
    vue({
      template: {
        compilerOptions: {
          isCustomElement: (tag) => tag.includes('-')
        }
      }
    })
  ]
}
```

**Vue Cli config:**
```js dark
// vue.config.js
module.exports = {
  chainWebpack: config => {
    config.module
      .rule('vue')
      .use('vue-loader')
      .tap(options => ({
        ...options,
        compilerOptions: {
          // treat any tag that starts with ion- as custom elements
          isCustomElement: tag => tag.startsWith('ion-')
        }
      }))
  }
}
```

> Exemplo de uso do componente `button` em **VUE**:

```javascript dark
<script setup>
  import { reactive } from 'vue'
  import '@olympus/button';
  const button = reactive({
    label: 'Botão no VUE',
    state: 'primary'
  })
</script>
<template>
  <ods-button
    :state="button.state"
  >
    {{button.label}}
  </ods-button>
</template>
<style>
  @import "@olympus/design-tokens/index.css";
</style>
```
