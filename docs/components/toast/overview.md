# Toast

## Usage

```html
<ods-toast title="Title"> Label </ods-toast>
```

```js
import '@olympus/components/toast'
```

> React

```jsx
import { OdsToast } from '@olympus/components/react/toast'
const App = () => <OdsToast> Label </OdsToast>
```
