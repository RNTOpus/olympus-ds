# Svelte

## Usage

::: warning Styles
Import **Olympus** tokens or rewrite tokens with your own values
:::

::: info Example
Button component
:::

```html
<script>
  import "@olympus/tokens/index.css";// [!code ++]
  import '@olympus/components/button';// [!code ++]
</script>
<ods-button />// [!code ++]
```
