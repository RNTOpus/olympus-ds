## CSS Parts

| Name          | Description                                              |
| ------------- | -------------------------------------------------------- |
| `wrapper`     | Access the `ods-field` custom component.                 |
| `field`       | The `<input>` or `<textarea>` element.                   |
| `label`       | The `<span>` field label. **Accessible by `ods-field`**  |
| `icon-button` | The right icon `<button>`. **Accessible by `ods-field`** |
