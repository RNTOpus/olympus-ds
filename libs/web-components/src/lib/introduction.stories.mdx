## Instalação

Para instalar `@olympus-ds/web-components` no seu projeto, rode o comando ([npm](https://www.npmjs.com/)):

```bash dark
npm install @olympus-ds/web-components
```

Se preferir [Yarn](https://yarnpkg.com/en/), rode:

```bash dark
yarn add @olympus-ds/web-components
```

### Uso:
```js dark
import '@olympus-ds/web-components';
```
> Caso o seu projeto não precise de todos os componentes do **ODS**, é recomendado a importação apenas do(s) componente(s) que for(em) utilizado(s), veja:
```js dark
import '@olympus-ds/web-components/lib/{nome-do-componente}';
```

> **Importante:** Para usar os tokens do **Olympus DS** será necessário importar o arquivo **index.css**, conforme exemplo mais abaixo:
- **OBS.:** É de responsabilidade da aplicação setar os tokens, desta forma é mais fácil para customizá-los.<br />
[Saiba mais sobre a biblioteca de Tokens do Olympus DS]().
```css dark
  @import "~@olympus-ds/design-tokens/index.css";
```

## Angular
> Para usar **Web Components** no Angular.<br />Adicione **CUSTOM_ELEMENTS_SCHEMA** no `@NgModule.schemas`, conforme exemplo abaixo:

```js dark
import { NgModule, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';
...
@NgModule({
  ...
  schemas: [CUSTOM_ELEMENTS_SCHEMA],
  ...
})
export class AppModule { }

```
> Exemplo de uso de um componente no **ANGULAR**:

```typescript dark
import '@olympus-ds/web-components/lib/button';

@Component({
  selector: 'my-component',
  template: `
    <ods-button
      [color]="buttonColor"
    >
      {{buttonText}}
    </ods-button>
  `,
  styleUrls: ['./my.component.css']
})
export class MyComponent {
  buttonColor = "primary"
  buttonText = "Botão no angular"
}
```

## Vue
> Para usar **Web Components** no Vue.<br />Adicione **isCustomElement** no `compilerOptions`, conforme exemplos abaixo:<br />[Documentação oficial](https://vuejs.org/guide/extras/web-components.html)

**In-Browser config:**
```js dark
app.config.compilerOptions.isCustomElement = (tag) => tag.includes('-')
```

**Vite config:**
```js dark
import vue from '@vitejs/plugin-vue'
export default {
  plugins: [
    vue({
      template: {
        compilerOptions: {
          isCustomElement: (tag) => tag.includes('-')
        }
      }
    })
  ]
}
```

**Vue Cli config:**
```js dark
// vue.config.js
module.exports = {
  chainWebpack: config => {
    config.module
      .rule('vue')
      .use('vue-loader')
      .tap(options => ({
        ...options,
        compilerOptions: {
          // treat any tag that starts with ion- as custom elements
          isCustomElement: tag => tag.startsWith('ion-')
        }
      }))
  }
}
```

> Exemplo de uso de um componente em **VUE**:

```javascript dark
<script setup>
  import { reactive } from 'vue'
  import '@olympus-ds/web-components/lib/button';
  const button = reactive({
    label: 'ODS Botão no VUE',
    color: 'primary'
  })
</script>
<template>
  <ods-button
    :color="button.color"
  >
    {{button.label}}
  </ods-button>
</template>
<style>
  @import "@olympus-ds/design-tokens/index.css";
</style>
```

## Svelte
> Não é necessário configuração para usar **Web Components** no Svelte, veja um exemplo de uso:

```javascript dark
  <script>
    import "@olympus-ds/design-tokens/index.css";
    import '@olympus-ds/web-components/lib/button';
    export let buttonText = "ODS Botão no Svelte";
    export let color = "primary"
  </script>
  <ods-button
    color={color}
    on:click="{() => color = 'success'}">
      {buttonText}
  </ods-button>
```
